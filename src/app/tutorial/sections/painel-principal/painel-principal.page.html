Painel Principal - Centro de Controle

O Painel Principal é o centro de controle do sistema N.D Connect. Aqui você tem uma visão geral de todos os leads, estatísticas em tempo real e acesso rápido às principais funcionalidades.

Leads Recentes

Visualização dos 5 leads mais recentes com:

  • Status visual com cores e ícones
  • Ações rápidas: ligar, email, WhatsApp, criar orçamento
  • Indicador de "não lido" para leads novos
  • Informações completas: nome, email, telefone, data

Botão Flutuante

No canto inferior direito, você encontra um botão flutuante com ícone "+" que permite:

  • Criar novo lead rapidamente
  • Acesso direto ao formulário de criação
  • Preenchimento automático dos dados

Atualização Automática

O painel se atualiza automaticamente:

  • A cada 30 segundos - dados sempre atualizados
  • Pull-to-refresh - arraste para baixo para atualizar
  • Notificações em tempo real - novos leads aparecem instantaneamente

Tutorial Integrado

No header do painel, você encontra:

  • Ícone de escola - acesso ao tutorial completo
  • Disponível apenas para dev/admin - controle de acesso
  • Guia interativo - aprenda enquanto usa

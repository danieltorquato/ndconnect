Catálogo de Produtos - Gestão Completa

O Catálogo de Produtos é onde você gerencia todo o portfólio de produtos e serviços da sua empresa, organizando preços, descrições e categorias.

Passos para Adicionar um Produto:

  1. Clique no botão "Adicionar Produto" na página de produtos
  2. Preencha as informações básicas:
    • Nome do produto (obrigatório)
    • Descrição detalhada
    • Categoria do produto
    • Preço unitário
  3. Configure informações adicionais:
    • Código do produto (SKU)
    • Unidade de medida
    • Estoque disponível
    • Status (ativo/inativo)
  4. Adicione imagens se disponível
  5. Clique em "Salvar" para adicionar ao catálogo

Dicas Importantes:

  • O nome do produto é obrigatório para salvar
  • Use descrições claras para facilitar a busca
  • Organize por categorias para melhor navegação
  • Mantenha preços atualizados regularmente

Como Buscar Produtos

Critérios de Busca:

Você pode buscar produtos por qualquer um destes campos:

  • Nome do produto
  • Descrição ou palavras-chave
  • Código do produto (SKU)
  • Categoria do produto
  • Preço (faixa de valores)

Como Buscar:

  1. Digite o termo na barra de pesquisa
  2. A busca é instantânea - resultados aparecem conforme você digita
  3. Use filtros por categoria para refinar resultados
  4. Combine busca + filtros para resultados precisos
  5. Limpe a busca para ver todos os produtos novamente

Como Editar Produtos

Atualize informações e preços dos produtos

 Manutenção

Como Editar:

  1. Localize o produto na lista do catálogo
  2. Clique no botão "Editar" (ícone de lápis)
  3. Modifique as informações desejadas
  4. Atualize preços, descrições ou categorias
  5. Clique em "Salvar" para confirmar as alterações

Dicas de Edição:

  • Preços atualizados refletem automaticamente nos orçamentos
  • Descrições claras melhoram a experiência do cliente
  • Categorias corretas facilitam a organização
  • Status ativo/inativo controla a visibilidade
